# 网站模板备份 (2026-02-18)

## 文件结构

```
├── index.html      # 首页
├── cases.html     # 案例视频页面 (9个视频)
└── case-gallery.html  # 案例展示页面 (4个锦旗图片)
```

## 首页布局 (固定结构)

1. **头部区域** - Logo + 标题 + 导航栏
2. **微信联系方式Banner** - 显眼位置显示微信号
3. **服务项目** - 3列卡片布局
4. **案例视频** - 3个视频 + 查看更多按钮
5. **案例展示** - 3个图片 + 查看更多按钮
6. **最新文章** - 3篇文章 + 查看更多按钮
7. **底部** - 版权信息 + 微信号
8. **悬浮微信按钮**

## 重要样式变量

```css
--primary: #b8964c        /* 金色主色 */
--bg-dark: #0d0d12       /* 深色背景 */
--bg-card: #16161f       /* 卡片背景 */
```

## 修改规则

✅ 可以修改：
- 内容信息（文字、图片链接）
- 颜色变量值
- 增加/删除模块
- 调整间距、数值

❌ 不能修改：
- 整体布局结构
- CSS类名
- 模块顺序
- 响应式断点
