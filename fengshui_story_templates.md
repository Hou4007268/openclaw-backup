# 风水故事模板库

## 模板1：客户案例型
**【标题】客户故事：XX问题，风水调整后XX变化**

**结构：**
1. 背景：客户基本情况/遇到的问题
2. 发现：风水勘察发现的XX问题
3. 调整：具体调整方案
4. 结果：调整后的变化（数据/感受）
5. 原理：背后的风水原理
6. 互动：提问引导

**示例：**
"上周看了一个办公室风水，客户说团队总吵架。
发现：办公桌正对尖角煞，白虎位过高。
调整：移动工位，加屏风，调整绿植。
结果：一周后反馈，沟通顺畅多了。
原理：尖角煞引发口舌，白虎主争斗。
你们遇到过类似问题吗？"

## 模板2：知识科普型  
**【标题】风水小知识：XX的讲究**

**结构：**
1. 现象：常见现象/误区
2. 原理：风水原理解释
3. 案例：实际案例佐证
4. 建议：实用调整建议
5. 互动：经验分享邀请

**示例：**
"很多人喜欢在门口放镜子。
原理：镜子属金，有反射作用，用错位置会反射走财气。
案例：有个客户门口放大镜子，生意一直不好，撤掉后好转。
建议：镜子不对门，不对床。
你家镜子放对了吗？"

## 模板3：对比实验型
**【标题】前后对比：调整XX前后的变化**

**结构：**
1. 调整前：问题描述/照片
2. 调整方案：具体改动
3. 调整后：变化描述/照片
4. 数据：如果有量化数据
5. 原理：为什么有效
6. 互动：猜猜哪里改了？

## 模板4：问答互动型
**【标题】粉丝提问：关于XX的风水问题**

**结构：**
1. 问题：粉丝真实问题
2. 分析：问题背后的风水逻辑
3. 建议：具体解决方案
4. 扩展：相关知识点
5. 互动：还有类似问题吗？

## 模板5：故事连载型
**【标题】风水日记：第X天，XX项目进展**

**结构：**
1. 项目背景
2. 当日发现
3. 调整过程
4. 初步感受
5. 下步计划
6. 互动：猜猜明天会发现什么？

## 内容质量控制标准

### 必须包含
1. 真实案例基础（可改编保护隐私）
2. 明确风水原理
3. 实用建议
4. 互动引导

### 避免内容
1. 迷信夸大
2. 绝对化表述
3. 隐私泄露
4. 专业术语堆砌
