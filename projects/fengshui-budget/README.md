# 💰 装修预算规划助手

基于 Ollama 本地模型开发的装修预算规划工具。

## 功能

- 输入房屋面积、户型、装修风格
- AI 生成详细预算分配方案
- 各空间预算明细
- 主材和家具建议
- 本地运行，不花 API 钱

## 技术栈

- **前端**: HTML + CSS + JavaScript
- **后端**: Python Flask
- **AI模型**: Ollama qwen2.5:latest

## 快速开始

```bash
pip install -r requirements.txt
python server.py
```

访问：`http://localhost:5003`

## 免费！

使用本地 Ollama 模型，无需 API Key。
