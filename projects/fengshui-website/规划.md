# 🏮 一宅一句 - 风水网站规划

## 网站定位

**Slogan**: 科学风水，AI赋能
**目标用户**: 装修党、买房族、对风水感兴趣的年轻人

---

## 功能模块

### 1. 首页
- 核心功能入口（卡片式）
- 每日风水小贴士
- 热门工具展示
- 案例展示

### 2. 户型分析（已有MVP）
- 上传户型图
- AI识别分析
- 风水问题诊断
- 调整建议

### 3. 风水配色（已有MVP）
- 房间类型选择
- AI推荐配色
- 风水寓意说明

### 4. 预算规划（已有MVP）
- 输入面积/户型/风格
- AI生成预算方案
- 分空间明细

### 5. 收纳规划（已有MVP）
- 房间/人口输入
- AI推荐收纳方案
- 断舍离建议

### 6. 风格测试（已有MVP）
- 5道趣味问答
- 测试最适合风格
- AI详细解读

### 7. 光照设计（已有MVP）
- 房间/朝向输入
- AI光照方案
- 节能tips

### 8. 知识库（待开发）
- 风水基础知识
- 装修避坑指南
- 常见问题FAQ

### 9. 案例展示（待开发）
- 业主案例分享
- 前后对比图
- 用户评价

---

## 技术架构

### 前端
- 单页应用（SPA）
- React 或 Vue
- 移动端优先

### 后端
- Python Flask 或 FastAPI
- 统一API网关
- Ollama 本地模型

### 部署
- Vercel（前端）
- 本地服务器（后端 Ollama）
- 或全部 Docker 化
