## 二、内容脚本

### 完整脚本：
```
【开场】（0-5秒）
画面：你出现在办公室场景中
配音："大家好，我是一宅一句"
字幕：办公室风水小课堂

【问题引入】（5-15秒）
画面：展示办公桌正对门的错误布局
配音："很多人的办公桌正对门口，这是风水大忌"
字幕：错误示范：办公桌对门

【问题分析】（15-25秒）
画面：用箭头示意气流的直冲
配音："这叫'冲煞'，气流直冲影响专注力和决策"
字幕：冲煞 - 气流直冲

【解决方案】（25-40秒）
画面：演示调整办公桌位置
配音："简单调整：办公桌侧放，避开直冲"
画面：展示调整后的正确布局
字幕：正确布局：侧放避冲

【额外建议】（40-50秒）
画面：在办公桌旁加一个小屏风或绿植
配音："可以加个小屏风或绿植，进一步化解"
字幕：加强：屏风/绿植

【结尾互动】（50-60秒）
画面：你面对镜头微笑
配音："你的办公桌有没有类似问题？评论区告诉我"
字幕：关注我，每天一个风水小知识
标签：#风水 #办公室风水 #视频 #一宅一句
```

### 精简版（如果时间有限）：
```
0-10秒：问题展示（办公桌对门）
10-25秒：解决方案演示（侧放调整）
25-35秒：额外建议（加屏风/绿植）
35-45秒：结尾互动
```

## 三、拍摄要求

### 场景设置：
1. **办公室环境**：
   - 简单的办公桌和椅子
   - 门口作为背景参考
   - 整洁有序的环境

2. **拍摄角度**：
   - 全景：展示整个布局
   - 中景：你的讲解画面
   - 特写：调整细节展示

3. **光线要求**：
   - 自然光或柔和的室内光
   - 避免背光或过暗

### 道具准备：
1. **必需道具**：
   - 办公桌和椅子
   - 门口（或模拟门口）
   - 手机/相机拍摄设备

2. **可选道具**：
   - 小屏风或隔断
   - 绿植（如发财树、绿萝）
   - 办公用品（电脑、文件夹等）

## 四、拍摄步骤

### 步骤1：错误布局拍摄（10分钟）
1. 设置办公桌正对门口
2. 拍摄全景展示错误布局
3. 你入镜讲解问题

### 步骤2：调整过程拍摄（10分钟）
1. 演示移动办公桌到侧面
2. 拍摄调整过程（可加速）
3. 展示调整后的正确布局

### 步骤3：加强效果拍摄（5分钟）
1. 添加屏风或绿植
2. 拍摄最终效果
3. 你讲解加强效果

### 步骤4：结尾拍摄（5分钟）
1. 面对镜头互动
2. 邀请评论和关注
3. 展示标签

## 五、剪辑要点

### 节奏控制：
- **开头**：快速吸引（5秒内出现核心问题）
- **中间**：清晰演示（每个步骤5-10秒）
- **结尾**：明确互动（留下行动指令）

### 字幕添加：
1. **关键点字幕**：错误示范、冲煞、正确布局等
2. **讲解字幕**：重要的讲解内容
3. **标签字幕**：视频结尾显示标签

### 背景音乐：
- 轻快的背景音乐（音量低于人声）
- 无版权音乐或简单音效
- 结尾音乐渐弱
