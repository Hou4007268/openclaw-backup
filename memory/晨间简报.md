# 晨间简报

> 每天开始工作时自动显示今日待办

---

## 触发时间

每天早上8:00（可配置）

## 简报内容

```markdown
🌅 晨间简报 | YYYY-MM-DD HH:MM

📋 今日待办：

【紧急重要】
- [ ] 任务1 | 截止时间
- [ ] 任务2 | 截止时间

【重要不紧急】
- [ ] 任务3
- [ ] 任务4

【日常事项】
- [ ] 任务5
- [ ] 任务6

📊 统计：
- 总待办：X项
- 今日到期：X项

❓ 你想从哪个任务开始？
```

## 数据来源

从 `memory/任务清单.md` 读取

## 触发条件

当用户进入目录时检查：
1. 是否有今日简报且未显示
2. 如果有，自动展示

## 预设问题

简报后自动询问：
- "今天想先做哪个？"
- "需要添加新任务吗？"
- "有什么优先级调整？"
