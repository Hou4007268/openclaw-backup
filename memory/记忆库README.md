# 记忆库 - 三层记忆架构

> 参考AI工作系统二代设计

## 目录结构

```
记忆库/
├── 情景记忆/    # 具体事件和对话（按月归档）
├── 语义记忆/    # 提炼的知识和模式
└── 强制规则/    # 系统行为约束
```

## 加载规则

| 类型 | 何时加载 |
|------|----------|
| 情景记忆 | 回溯对话时 |
| 语义记忆 | 相关任务时 |
| 强制规则 | 始终加载 |

## 文件命名规范

- 情景记忆：`YYYY-MM-DD-主题.md`
- 语义记忆：`[主题]_[类型].md`
- 强制规则：`强制规则_[领域].md`
