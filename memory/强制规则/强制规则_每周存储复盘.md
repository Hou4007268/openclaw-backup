# 每周存储复盘

> 每周一自动检查文件位置是否正确

---

## 触发时间

每周一晨间简报时

## 执行内容

```bash
1. 检查错位文件
2. 统计错位数量
3. 生成复盘报告
```

## 检查规则

| 检查项 | 规则 |
|--------|------|
| 推文草稿 | 必须在 drafts/twitter-*.md |
| 小红书笔记 | 必须在 drafts/xiaohongshu-*.md |
| 文章 | 必须在 outputs/articles/ |
| 配图 | 必须在 outputs/images/ |
| 临时文件 | 必须在 artifacts/ |
| 系统文件 | 必须在根目录或memory/ |

## 输出格式

```markdown
## 每周存储复盘 | YYYY-MM-DD

### 错位文件
- [文件路径] → 应在 [正确路径]

### 统计
- 总文件数：X
- 错位文件：X
- 准确率：X%

### 建议
- [优化建议]
```

## 执行命令

```bash
# 手动触发
qmd query "storage review"

# 或
# 每周一自动执行
```
