# 风水内容创作工具包

## 一、快速内容生成模板

### 模板1：客户故事型（已验证有效）
```
【场景】今天见了个[行业]客户
【问题】"[客户原话描述问题]"
【观察】我一看/一听，发现[风水问题]
【建议】"[你的专业建议]"
【原理】因为风水上[位置/物品]代表[含义]
【幽默】[客户反应/你的内心OS]
【标签】#风水 #玄学 #[场景标签] #[风格标签]
```

**填充示例**：
```
【场景】今天见了个开餐厅的客户
【问题】"大师，我店里生意一直不好"
【观察】我一看，收银台正对厕所门
【建议】"你这收银台得挪位置，不能对着污秽之地"
【原理】因为风水上财位要干净，厕所是污气出口
【幽默】客户："难怪我数钱都闻到味道"
【标签】#风水 #商铺风水 #财运 #真实故事
```


## 五、内容素材库

### 客户案例分类：
1. **住宅问题**：
   - 睡眠不好（卧室布局）
   - 家庭矛盾（客厅/厨房）
   - 孩子学习（书房/儿童房）
   - 健康问题（卫生间/阳台）

2. **办公室问题**：
   - 团队不和谐（会议室/公共区）
   - 业绩下滑（老板办公室/销售区）
   - 员工流失（工位布局）
   - 决策失误（管理层办公室）

3. **商业问题**：
   - 生意不好（店面布局）
   - 客户流失（接待区）
   - 资金紧张（财务室）
   - 竞争压力（外部环境）

### 风水知识点库：
1. **基础原理**：
   - 阴阳平衡
   - 五行相生相克
   - 八卦方位
   - 气场流动

2. **常见禁忌**：
   - 门对门冲煞
   - 横梁压顶
   - 镜子对床
   - 厕所居中

3. **化解方法**：
   - 屏风隔断
   - 植物调节
   - 颜色搭配
   - 物品摆放

## 六、快速创作流程

### 步骤1：选择主题（1分钟）
- 从素材库选一个案例或知识点
- 确定目标受众（住宅/办公室/商业）
- 选择内容形式（视频/图文）

### 步骤2：套用模板（3分钟）
- 选择合适的内容模板
- 填充具体内容
- 优化开头吸引力

### 步骤3：设计互动（2分钟）
- 添加提问或投票
- 设计回复引导
- 设置互动奖励

### 步骤4：优化标签（1分钟）
- 核心标签必加
- 场景标签匹配
- 风格标签点缀

### 步骤5：检查发布（1分钟）
- 检查错别字
- 预览效果
- 定时发布

**总时间**：8分钟内完成一条内容

## 七、批量创作技巧

### 周末批量创作：
1. **周六上午**：规划下周主题（7个）
2. **周六下午**：撰写所有文案（套用模板）
3. **周日上午**：准备图片/视频素材
4. **周日下午**：设置定时发布

### 素材复用：
1. **热门内容**：3个月后优化重发
2. **系列内容**：整理成合集
3. **粉丝问答**：整理成专题
4. **节日热点**：提前准备相关内容
