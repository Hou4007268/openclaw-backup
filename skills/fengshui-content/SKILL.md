# 风水内容创作技能

> 为"一宅一句"品牌定制，去除AI味，写出真人风格

---

## 核心原则（去AI味）

### 1. 像真人一样说话
- 口语化，不要书面语
- 用短句，不用长句
- 有语气词（😂、啊、呢、嘛）
- 像和朋友聊天，不像写报告

### 2. 有个性有态度
- 有观点，不中立
- 敢说 YES or NO
- 有情绪表达（惊讶、吐槽、分享）

### 3. 用故事不用道理
- 用客户案例讲故事
- 不要干巴巴讲知识
- 细节越多越好玩

---

## 禁止清单（去除AI味）

### 废话开场白 ❌
禁止：
- "有一说一"
- "说实话"
- "不得不說"
- "其实"
- "总体来说"
- "综上所述"

### 商业黑话 ❌
禁止：
- "赋能"
- "闭环"
- "抓手"
- "打法"
- "痛点"
- "赛道"
- "壁垒"

### 公式化句式 ❌
禁止：
- "首先……其次……最后"
- "一方面……另一方面"
- "我认为……"
- "值得注意的是……"

### AI常用词 ❌
禁止：
- "深入浅出"
- "为XX赋能"
- "助力XX"
- "让我们一起"
- "相信大家"
- "你值得拥有"

## 正确示范 ✅

| AI写法 | 真人写法 |
|--------|----------|
| 我认为这个观点 | 这事我太有发言权了 |
| 首先其次最后 | 我再补充一点 |
| 综上所述 | 总之记住这句 |
| 赋能 | 让XX更强 |
| 助力 | 帮忙 |

## 内容模板（4种已验证有效）

### 模板1：客户故事型（最常用）

```
【场景】今天见了个[行业]客户
【问题】"[客户原话]"
【观察】我一看，发现[风水问题]
【建议】"[建议]"
【原理】因为[位置/物品]代表[含义]
【幽默】[客户反应/内心OS] 😂
【互动】[引导评论]
【标签】#风水 #玄学 #一宅一句
```

### 模板2：知识问答型

```
【提问】很多人问：[常见问题]
【误区】以为：[错误认知]
【真相】其实：[正确解释]
【建议】正确做法：[具体]
【互动】你们家有过类似情况吗？
```

### 模板3：对比纠正型

```
【标题】[常见做法] VS [正确做法]
【错误】这样做：[描述]
【问题】会导致：[后果]
【正确】应该：[方法]
【提醒】还有一点要注意
```

### 模板4：趣味故事型

```
【开头】笑死，今天遇到个奇葩客户
【发展】对方说："[有趣的问题]"
【转折】我：？？？大哥，我是看风水的
【科普】顺便说句：[知识点]
【结尾】专业的事找专业的人
【互动】你们遇到过啥奇葩问题？
```

## 标签策略

### 必带
```
#风水 #玄学 #一宅一句
```

### 场景标签（选2-3）
- 住宅：#卧室 #客厅 #厨房 #阳台
- 办公：#办公室风水 #事业 #财运
- 商业：#商铺风水 #店面 #招财

### 风格标签（选1-2）
- #姜还是老的辣 #真实故事
- #知识 #科普 #传统文化

## 快速创作流程

1. **选主题** - 从素材库选案例
2. **套模板** - 按结构填充
3. **去AI味** - 检查禁止清单
4. **加互动** - 引导评论
5. **验标签** - 核心标签必加
