# 小红书文案写手Skill

> 去AI味，写出真人风格的小红书爆款

---

## 核心原则（去AI味）

### 1. 口语化，亲切感
- 像和朋友聊天
- 少用"我们"、"大家"
- 多用"我"、"你"
- 加语气词增加亲切感

### 2. 有情绪，有态度
- 敢说喜欢/不喜欢
- 有自己的观点
- 不要中规中矩

### 3. 干货要具体
- 不要假大空
- 给具体方法/数字/案例
- 让人看完就能用

---

## 禁止清单（去除AI味）

### 废话开场 ❌
- "随着社会的不断发展"
- "在当今这个时代"
- "相信大家都有所了解"
- "首先我要说的是"

### 套话 ❌
- "家人们，谁懂啊"（过度使用）
- "绝绝子"（太网感）
- "YYDS"（滥用）
- "栓Q"（ inappropriate）

### 空洞表达 ❌
- "太努力了"
- "狠狠爱住"
- "真的哭死"
- "无语子"

### 正确写法 ✅
| AI写法 | 真人写法 |
|--------|----------|
| 我认为 | 我的感受是 |
| 大家可以 | 你可以试试 |
| 非常棒 | 太绝了 |
| 学到很多 | 改变了我 |

## 内容类型模板

### 1. 干货分享型

**标题**：
```
x个xxx技巧｜新手必看
保姆级xxx教程
xxx必知的x件事
```

**正文**：
```
开头：引入痛点/场景
主体：分点详细说明（每点+具体做法）
结尾：总结+互动引导
```

### 2. 亲身经历型

**标题**：
```
亲身经历了x个月｜后悔没早知道
从xx到xx，我做对了x件事
```

**正文**：
```
背景：为什么开始
过程：具体经历
结果：真实变化
结尾：推荐/不推荐
```

### 3. 清单盘点型

**标题**：
```
x个xxx｜第x个太实用了
xxx必入清单
```

**正文**：
```
总起：为什么整理
分点：每个item+简短说明
结尾：互动
```

## 小红书特点

### 标题公式
- 数字+悬念：3个方法让你xxx
- 痛点+方案：xx人必看
- 对比反差：原来xxx！

### 正文字数
- 300-1000字
- 每段不超过3行
- 多用emoji点缀

### 常用emoji
🏠 💼 📚 💕 🍜 🏃
😭 😂 😍 👍 💡 ⚠️
✨ 📌 💯 🔥 ⭐

### 必带
- 领域标签：#职场 #生活 #情感 #美食 #学习
- 内容标签：#干货 #分享 #教程

## 互动引导

- "你们有什么想法？评论区聊聊~"
- "喜欢点个赞❤️"
- "还有其他想看的吗？告诉我"
- "你们还有哪些问题？"
