# 小红书视觉内容策划Skill

> 将文字内容拆解为小红书风格系列信息图

---

## 激活条件

**当用户说：**
- "小红书配图"
- "生成笔记图片"
- "发小红书"
- 在#📕-小红书频道发布内容

---

## 核心规范

### 比例尺寸
| 类型 | 比例 |
|------|------|
| 竖版（推荐） | 3:4 或 9:16 |
| 横版 | 16:9 |

### 视觉风格
- **类型**：信息图（Infographic）
- **风格**：卡通/手绘风格
- **背景色**：莫兰迪色系

---

## 拆解结构

| 位置 | 内容 |
|------|------|
| 封面图 | 视觉冲击 + 核心标题 |
| 内容图 | 每张1个核心观点 |
| 结尾图 | 总结 / 行动号召 |

---

## 输出格式

```markdown
### 第 X 张
**定位**：封面/内容/结尾
**核心信息**：xxx

**视觉提示词**：
小红书风格信息图，竖版（3:4），卡通风格，[背景色]背景...
```
