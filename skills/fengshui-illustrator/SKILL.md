- "配图"

- "发推文"

---

## 风格库

### 1. warm（温暖亲和）- 推荐
- **配色**：暖黄色、金色、棕色、橙色
- **元素**：传统中式纹样、灯笼、红包、富贵竹
- **适用**：客户故事、温馨提醒、居家风水

### 2. playful（趣味涂鸦）
- **配色**：明亮活泼
- **元素**：简笔画风格、卡通人物
- **适用**：趣味知识、互动话题

### 3. minimal（极简）
- **配色**：黑白灰、点睛色
- **元素**：线条、几何图形
- **适用**：知识科普、干货分享

### 4. notion（线稿风）
- **配色**：白色背景、黑色线条
- **元素**：手绘线条、示意图
- **适用**：户型分析、布局教程

### 5. traditional（传统中国风）
- **配色**：中国红、墨黑、金色
- **元素**：祥云、青铜器、水墨画
- **适用**：传统节日、文化解读

---

## 自动选择规则

| 关键词 | 风格 |
|--------|------|
| 案例、客户、故事 | warm |
| 趣味、搞笑 | playful |
| 知识、科普 | minimal |
| 户型、布局 | notion |
| 传统、节日 | traditional |

---

## 工作流程

1. 分析内容 → 2. 识别配图位置 → 3. 选择风格 → 4. 生成提示词 → 5. 调用生图 → 6. 输出

---

## 调用生图工具

### 松耦合原则
不要写死调用哪个工具，描述需要什么能力即可。

### 推荐写法
```
1. 检查可用的画图Skill
2. 如果有多个，选择最合适的
3. 如果用户指定了，按用户选择
4. 调用时传递：风格 + 内容描述
```

### 实际调用示例
```
用户：帮我画一张卧室风水图
AI执行：
  1. 检查可用画图工具（xhs-images/其他）
  2. 生成提示词
  3. 调用画图工具
  4. 返回图片路径
```
