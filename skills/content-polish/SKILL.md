## 检查项目

### 1. 标点符号检查
- [ ] 中文内容使用全角标点（，。！？""）
- [ ] 英文和数字两侧有空格
- [ ] 省略号用……不用...

### 2. 错别字检查（常见）
- [ ] "的"vs"得"vs"地"
- [ ] "您"vs"你"
- [ ] 常见错别字

### 3. 格式检查
- [ ] 段落间有空行
- [ ] 标题格式正确
- [ ] 列表编号统一

### 4. AI味检查
- [ ] 无废话开头
- [ ] 无套话
- [ ] 无过度格式化

---

## 自动修正（脚本可做）

```bash
# 标点符号统一
sed -i 's/,/，/g' file.md
sed -i 's/\./。/g' file.md

# 英文数字两侧加空格
sed -i 's/\([a-zA-Z0-9]\)/ \1 /g' file.md

# 删除多余空行
cat -s file.md
```
