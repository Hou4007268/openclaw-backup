## 采集频率建议

| 类型 | 频率 | 说明 |
|------|------|------|
| 热点 | 2小时/次 | 及时捕捉热点 |
| 竞品 | 1次/天 | 分析爆款规律 |
| 资讯 | 3次/天 | 行业动态 |

## 数据存储

采集的数据存储在：

```
output/
├── hot/
│   ├── weibo_2024-01-15.json
│   ├── zhihu_2024-01-15.json
│   └── ...
├── competitor/
│   ├── 混知_2024-01-15.json
│   └── ...
└── news/
    ├── 36kr_2024-01-15.json
    └── ...
```

## 数据格式

```json
{
  "title": "文章标题",
  "content": "正文内容(摘要)",
  "url": "原文链接",
  "source": "来源",
  "author": "作者",
  "publish_time": "发布时间",
  "read_count": "阅读量(如有)",
  "like_count": "点赞数(如有)",
  "comment_count": "评论数(如有)",
  "tags": ["标签1", "标签2"],
  "crawl_time": "采集时间"
}
```
