# 大纲生成器Skill

> 一次生成3个不同风格的大纲，像抽卡一样选择

---

## 激活条件

**用户说：**
- "写大纲"
- "生成大纲"
- "帮我列一下"
- "不知道怎么写"

---

## 核心功能

一次生成**3个不同风格**的大纲，让用户像抽卡一样选择：

| 版本 | 结构风格 | 适用场景 |
|------|----------|----------|
| A版 | 时间线叙事 | 故事型、经历分享 |
| B版 | 问题-解决方案 | 干货科普、教程 |
| C版 | 观点碰撞 | 争议话题、评论分析 |

---

## 输出格式

```markdown
## 大纲A：时间线叙事
### 结构
1. 开场：xxx
2. 背景：xxx
3. 发展：xxx
4. 高潮：xxx
5. 结尾：xxx

## 大纲B：问题-解决方案
### 结构
1. 问题引入：xxx
2. 原因分析：xxx
3. 解决方案1：xxx
4. 解决方案2：xxx
5. 总结：xxx

## 大纲C：观点碰撞
### 结构
1. 观点A：xxx
2. 观点B：xxx
3. 碰撞分析：xxx
4. 结论：xxx
5. 行动建议：xxx
```

---

## 工作流程

1. **理解主题** - 分析用户需求
2. **生成A版** - 时间线叙事风格
3. **生成B版** - 问题解决风格
4. **生成C版** - 观点碰撞风格
5. **让用户选择** - 不满意可重开

---

## 使用示例

**用户：** "帮我写一篇关于卧室风水的文章大纲"

**输出：**
```
## 大纲A：时间线叙事
1. 我第一次去客户家看卧室
2. 发现床摆放问题
3. 调整后的变化
4. 客户的反馈

## 大纲B：问题-解决方案
1. 卧室常见问题有哪些
2. 为什么这些问题影响睡眠
3. 5步调整建议
4. 注意事项

## 大纲C：观点碰撞
1. 有人说不信风水
2. 科学的解释
3. 信与不信的平衡点
4. 实用建议
```

---

## 后续步骤

用户选择大纲后：
- 如果选A → 用fengshui-content按时间线写
- 如果选B → 用干货模板写
- 如果选C → 用对比模板写
