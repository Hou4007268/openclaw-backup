# 风水小红书创作技能

为"一宅一句"品牌定制的小红书内容创作技能。

## 功能
- 生成风水相关小红书笔记
- 标题生成（带emoji、数字、悬念）
- 正文创作（分段清晰、emoji丰富）
- 标签建议
- 配图建议

## 激活场景
用户提到：
- 小红书笔记
- 发笔记
- 写笔记
- xiaohongshu
- 发一篇风水

## 内容方向
- 租房风水避坑
- 客厅沙发摆放
- 卧室床位讲究
- 装修风水禁忌
- 真实风水故事
- 每日风水小贴士

## 输出格式
```
标题：[标题]

正文：
[正文内容]

#标签1 #标签2 #标签3

配图建议：
- [图片类型1]
- [图片类型2]
```

## 风格
- 轻松幽默
- 实用干货
- emoji丰富
- 结尾引导评论
