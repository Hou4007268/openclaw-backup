# 风水小红书创作技能

> 为"一宅一句"品牌定制的小红书内容创作技能

## 用于
- 生成风水相关小红书笔记
- 标题生成（带emoji、数字、悬念）
- 正文创作（分段清晰、emoji丰富）
- 标签建议
- 配图建议

## 不用于
- 推特内容（用 fengshui-content 技能）
- 微信公众号（用 md2wechat 技能）
- 纯学术风水论文
- 超过1000字的超长笔记（小红书偏好短内容）

## 激活场景
用户提到：
- 小红书笔记
- 发笔记
- 写笔记
- xiaohongshu
- 发一篇风水

# ⚠️ 内容规范（必须遵守！）

## 绝对禁止
1. **不要留任何联系方式** - 微信号、手机号、QQ号、邮箱
2. **不要写网站链接** - 12zn.com、任何网址
3. **不要提"风水师"身份** - 只能说"爱好者"、"爱好者分享"
4. **不要提"算命"、"预测"** - 只能说"科普"、"文化"
5. **不要用"招财"、"转运"等营销词** - 只能说"建议"、"注意"
6. **不要写"效果保证"、"一定有效"** - 只能说"供参考"

## 合规表达
| 违规词 | 合规替代 |
|--------|----------|
| 加微信 | 评论区聊聊 |
| 风水师 | 风水爱好者 |
| 算命 | 传统文化科普 |
| 招财转运 | 家居建议 |
| 一定有效 | 可以试试 |
| 看运势 | 分享知识 |

## 简介合规写法
```
🏠 家居风水科普
💡 帮你避开装修坑
✨ 科学看待风水
```

## 笔记结尾引导（合规）
- "你们家有这个问题吗？评论区说说~"
- "你们家沙发摆对了吗？发图帮你看~"
- "有问题评论区聊"
