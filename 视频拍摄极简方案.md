# 视频拍摄极简方案（2月8日10:00发布）

## 核心原则：最快最简单的方式完成

## 一、最低要求版本（5分钟拍摄）

### 场景要求：
- 一张桌子（办公桌、书桌、餐桌都可）
- 一个门或门口的概念
- 你的手机和自拍杆/支架

### 拍摄步骤：

**步骤1：问题展示（1分钟）**
- 手机固定，你入镜
- 指着桌子："很多人办公桌这样摆，正对门口"
- 简单手势示意"直冲"的问题

**步骤2：解决方案（2分钟）**
- 移动桌子到侧面（或用手势示意）
- 讲解："简单调整，侧放避开直冲"
- 补充："可以加个屏风或绿植更好"

**步骤3：结尾互动（2分钟）**
- 面对镜头微笑
- 说："你的办公桌有没有类似问题？评论区告诉我"
- 结束语："关注我，每天一个风水小知识"

### 剪辑要求：
- 用手机剪辑APP（剪映、快影等）
- 只保留核心内容，剪掉多余部分
- 添加关键字幕：办公桌对门、冲煞、侧放调整
- 加轻快的背景音乐

## 二、口播版本（无需实景）

### 拍摄方式：
- 你面对镜头直接讲解
- 背景干净即可（白墙、书架等）
- 用手势配合讲解

### 脚本（45秒）：
```
0-10秒：大家好，我是一宅一句。很多人办公桌摆错了位置。
10-25秒：正对门口是风水大忌，这叫"冲煞"，影响专注力。
25-35秒：简单调整：办公桌侧放，避开直冲。可以加屏风或绿植。
35-45秒：你的办公桌有没有类似问题？评论区告诉我。关注我，每天一个风水小知识。
```

## 三、图文动画版本（无需出镜）

### 制作方式：
1. 用PPT做3张图：
   - 图1：办公桌正对门（错误）
   - 图2：办公桌侧放（正确）
   - 图3：加屏风/绿植（加强）

2. 录制配音：
   - 按上面脚本录音
   - 清晰、有节奏

3. 合成视频：
   - 图片+配音+字幕
   - 简单转场效果

## 四、最简执行流程

### 时间安排：
- **拍摄**：5-10分钟（任一方案）
- **剪辑**：10-15分钟（手机APP）
- **导出上传**：5分钟
- **总时间**：20-30分钟

### 设备要求：
- 手机（能拍视频）
- 手机支架或靠墙放置
- 安静环境（录音清晰）

### 内容核心：
1. **问题**：办公桌对门不好
2. **解决**：侧放调整
3. **互动**：提问引导评论
4. **价值**：实用风水知识
