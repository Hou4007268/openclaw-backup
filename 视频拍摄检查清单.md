# 视频拍摄检查清单 - 2月8日办公室风水视频

## 一、拍摄前准备（今晚/明早）

### 设备检查：
- [ ] 手机/相机电量充足
- [ ] 存储空间足够（至少2GB空闲）
- [ ] 手机支架或稳定设备
- [ ] 备用充电宝

### 环境准备：
- [ ] 办公室场景清理整洁
- [ ] 办公桌和椅子位置可调整
- [ ] 门口区域清晰可见
- [ ] 光线充足（自然光或补光）

### 道具准备：
- [ ] 办公用品（电脑、文件夹等）
- [ ] 小屏风或隔断（可选）
- [ ] 绿植（发财树、绿萝等，可选）
- [ ] 拍摄脚本打印或电子版

## 二、拍摄流程检查

### 步骤1：错误布局拍摄（10分钟）
- [ ] 办公桌正对门口摆放
- [ ] 拍摄全景展示错误布局
- [ ] 你入镜讲解问题（面对镜头）
- [ ] 录制清晰讲解音频

### 步骤2：调整过程拍摄（10分钟）
- [ ] 演示移动办公桌到侧面
- [ ] 拍摄调整过程（可分段拍摄）
- [ ] 展示调整后的正确布局
- [ ] 讲解调整原理

### 步骤3：加强效果拍摄（5分钟）
- [ ] 添加屏风或绿植（如有）
- [ ] 拍摄最终效果展示
- [ ] 讲解加强效果的作用
- [ ] 展示整体改善

### 步骤4：结尾互动拍摄（5分钟）
- [ ] 面对镜头微笑互动
- [ ] 清晰说出互动问题
- [ ] 展示标签和关注引导
- [ ] 保持自然亲切的表情

## 三、拍摄技巧提醒

### 画面质量：
- [ ] 横屏拍摄（16:9比例）
- [ ] 分辨率1080p或以上
- [ ] 对焦清晰，不模糊
- [ ] 画面稳定，不抖动

### 音频质量：
- [ ] 环境安静，无背景噪音
- [ ] 说话清晰，音量适中
- [ ] 测试录音效果
- [ ] 避免回声

### 讲解技巧：
- [ ] 语速适中，不慌不忙
- [ ] 表情自然，有亲和力
- [ ] 手势配合讲解
- [ ] 与镜头有眼神交流

## 四、备用方案准备

### 如果无法实景拍摄：
- [ ] 准备PPT或图片素材
- [ ] 录制配音讲解
- [ ] 使用简单动画效果
- [ ] 确保内容清晰传达

### 如果时间有限：
- [ ] 精简脚本到30秒
- [ ] 重点展示核心问题
- [ ] 简化调整步骤
- [ ] 保证关键信息完整

## 五、剪辑要点检查
