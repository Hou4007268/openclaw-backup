## 使用方法

每次调用AI后，记录：
- 日期时间
- 模型
- 任务类型
- 输入tokens
- 输出tokens
- 预估费用

## 记录格式

```
## 2026-02-18

### 推文创作
- Kimi: 8k输入 / 2k输出 = ¥0.05
- 耗时: 45s

### 合议模式
- DeepSeek x3角色: 3k x 3 = 9k输入 / 1k x 3 = 3k输出 = ¥0.12
- 耗时: 2m

### 小红书配图
- MiniMax: 5k输入 / 3k输出 = ¥0.08
- 耗时: 30s

---

**今日总计**: ¥0.25
```

## 自动追踪

在 MEMORY.md 中维护每日的费用汇总。

## 查看命令

说"查看费用"或"本月花了多少"时，自动汇总并报告。
