# 合议模式 - 多角色决策系统

> 让AI扮演不同角色辩论，决策质量翻倍

## 使用方法

当需要做重要决策时，使用以下prompt模板：

```
我想请教你一个问题：[你的问题]

请分别从以下3个角度分析：

1. 【乐观派】战略顾问 - 看好这个方向的理由有哪些？机会点？
2. 【悲观派】风险预警员 - 可能有哪些问题？最坏的情况？
3. 【实操派】落地专家 - 具体执行步骤是什么？需要什么资源？

最后，综合以上三个视角，给出一个平衡的建议。
```

## 快速调用

### 风水内容选题
```
【乐观派】这个话题的传播潜力在哪里？
【悲观派】有哪些风险要避开？
【实操派】怎么写更容易火？

最后综合建议。
```

### 投资/花销决策
```
【乐观派】这笔支出的回报预期？
【悲观派】可能有哪些隐藏成本？
【实操派】最小可行性方案？

最后综合建议。
```

### 技术选型
```
【乐观派】这个技术方案的优势？
【悲观派】潜在坑和迁移成本？
【实操派】怎么逐步迁移？

最后综合建议。
```

---

## 自动触发

当用户说"分析一下"、"帮我看看"、"哪个好"、"建议"等决策类词汇时，自动使用合议模式。
