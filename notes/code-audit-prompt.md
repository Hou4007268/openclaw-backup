# 代码审计 prompt - 四视角审查

请从以下四个角度审查我工作区的代码（/Users/yachaolailo/projects/openclaw-backup）：

## 1. 攻击者视角 🔴
- 查找安全漏洞：SQL注入、XSS、命令注入
- 敏感信息泄露：API密钥、密码、token
- 不安全的依赖

## 2. 防御者视角 🛡️
- 错误处理是否完善
- 权限检查是否到位
- 日志记录是否充分

## 3. 隐私官视角 🔒
- 数据收集是否过度
- 是否有用户数据泄露风险
- 隐私政策合规性

## 4. 运维视角 ⚙️
- 代码是否有性能问题
- 是否有内存泄漏风险
- 依赖是否过时

## 输出格式

```

```
